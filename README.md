# Mensuras Corporis (old APK)

**Note:** This is an old prototype and does not reflect the final version I am currently developing.

## About the project

Mensuras Corporis was an initial idea for a free open-source tool designed to help healthcare and physical education professionals perform accurate and reliable anthropometric assessments.

However, I didn't like the result – it was bad. So I decided to rebuild it from scratch.

## The new project: OSAA

The new project is called **OSAA (Open Source Anthropometric Application)**. I am developing it locally (not uploaded to the cloud yet), but it is already well advanced. I am studying anthropometric assessments and evaluation protocols to finish the back-end.

**Planned features:**
- PWA application – after first access, it will work 100% offline
- APK versions to make installation easier on mobile devices (for those unfamiliar with browsers)

# Mensuras Corporis (APK antigo)

**Atenção:** Este é um protótipo antigo e não reflete a versão final que estou desenvolvendo.

## Sobre o projeto

O Mensuras Corporis foi uma ideia inicial de uma ferramenta open-source gratuita, desenvolvida para auxiliar profissionais da saúde e da educação física na realização de avaliações antropométricas precisas e confiáveis.

Porém, não gostei do resultado – ficou ruim. Então decidi refazer do zero.

## O novo projeto: OSAA

O novo projeto se chama **OSAA (Open Source Anthropometric Application)**. Estou desenvolvendo localmente (ainda não subi para a nuvem), mas já está bem adiantado. Estou estudando sobre avaliações antropométricas e protocolos de avaliação para finalizar o back-end.

**Características planejadas:**
- Aplicação PWA – após o primeiro acesso, funcionará 100% offline
- Versões em APK para facilitar a instalação em dispositivos móveis (para quem não tem familiaridade com navegadores)
